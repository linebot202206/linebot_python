## 說明

對機器人，或是在有機器人的群組，輸入「help」取得說明。

說明同下：

```text
輸入以下指令：
- 猜數字：1A2B
- 猜數字2：1A2B 強化版
- 比大小：2 個人輪流輸入「骰」來取得點數
- 比大小 N：N 為數字，N 個人輪流輸入「骰」來取得點數
- 抽：隨機出現圖片
- 抽正妹：隨機正妹
- 抽星座：隨機星運圖
- 抽美食：隨機美食
- BTS：隨機取得 BTS 照片
```

更多功能持續開發中，歡迎提交 issue 許願。

## 開發

```sh
git clone https://github.com/tigercosmos/angel.git
cd angel
npm install

npm start # run server
npm run console # console mode
```
